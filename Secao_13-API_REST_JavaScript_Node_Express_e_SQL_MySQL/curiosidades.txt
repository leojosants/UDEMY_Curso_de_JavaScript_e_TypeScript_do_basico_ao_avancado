-> Nodemon:
    Serve para reiniciar o servidor automaticamente a cada modificação salva no código e nos ajuda a ganhar tempo.

-> Sucrase:
    Utiliza a sintaxe do ES6 na importação e exportação de módulos, a mesma sintaxe usada no React.

-> ESLint:
    É um linter, uma ferramenta de análise estática, para as linguagens Javascript e Typescript, sendo o mais popular do mundo para tais linguagens. criado por Nicholas C. Zakas em 2013. As regras do ESLint são customizáveis, cobrindo tanto problemas de qualidade de software como de estilo.
